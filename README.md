# Sergio J. Peralta | Modern AI-Driven Portfolio

A high-performance, modern, and animated portfolio website built with React, Vite, Tailwind CSS, and Framer Motion. 

## ✨ Key Features

- **Modern Tech Stack**: React 18, Vite, Tailwind CSS, and Framer Motion for premium animations.
- **AI-Driven Branding**: Professional identity focused on AI solutions and enterprise SaaS architectural excellence.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Glassmorphism UI**: Sleek dark-themed aesthetic inspired by industry leaders like Stripe and Linear.
- **Dynamic Projects**: Interactive project cards with direct links to live demonstrations.
- **Direct Scheduling**: Integrated Google Calendar scheduling for seamless meetings.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (or yarn/pnpm)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components/`: Reusable React components (Navbar, Hero, Experience, etc.)
- `src/data/`: Centralized portfolio data model.
- `src/index.css`: Tailwind directives and global glassmorphism styles.
- `public/`: Static assets.
